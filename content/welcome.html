Welcome to Imgur Uploader for Firefox!

Thanks for installing the official Imgur Firefox extension.

Use Imgur Uploader to:

Rehost Images

Right click on any image and select Imgur → Quick Upload.

Screenshot Websites

Right click on any web site and select Imgur → Upload Screenshot. The current view of the page will be uploaded. To screenshot a specific part of the page, select Imgur → Upload Selective Screenshot.

Upload from the Clipboard

Right click on any web site and select Imgur → Upload from Clipboard.
